## 需求描述

记录“干瞪眼”游戏的分数以及输赢金额。过程中可以自己设置一些基础信息，包括每张牌的所值的价格；炸弹💣和全关是否翻倍。 单纯是为了大家一起玩的时候能更好的记账😂
